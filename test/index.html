Система для обслуживания сайтов, основанных на статических HTML-файлах. Ориентирована на небольшие сайты, требующие периодического обновления. Все изменения на сайте сохраняются прямо в HTML-файлах, что избавляет от необходимости иметь сервер базы данных. Не требует какой-либо интеграции с сайтом, достаточно разместить каталог с системой в корневой директории сайта. Система автоматически сканирует структуру сайта и предоставляет удобный доступ ко всем файлам через веб-интерфейс.

Основной особенностью Textolite является визуальный редактор с возможностью редактировать содержимое сайта без каких-либо форм ввода, достаточно кликнуть по элементу на сайте. Для более серьезных изменений предусмотрен редактор исходного кода с подсветкой синтаксиса и нумерацией строк. Поддерживаются файлы с расширением html, htm, shtml, shtm, stm, xml, php, js, css. Для загрузки или удаления файлов есть удобный файловый менеджер с возможностью многопоточной пакетной загрузки файлов на сервер.

Лицензионное соглашение

Система распространяется бесплатно. Все права принадлежат команде разработчиков Textolite. Пользователь имеет право использовать систему по прямому назначению на любых проектах, в том числе коммерческих. Допускается создание альтернативных ссылок на оригинальный архив с системой или прямая передача третьим лицам. Пользователь не имеет право:

  • Вносить какие-либо изменения в исходный код системы.
  • Распространять или использовать модифицированные копии системы.
  • Создавать собственные продукты, основанные на исходном коде системы.
  • Распространять или использовать исходные файлы или какие-либо части системы отдельно.
  • Переименовывать, изменять или удалять копирайты системы.

Использование системы означает полное согласие с условиями данного соглашения.

Установка

Для нормальной работы системы нужен веб-сервер Apache с PHP версии не ниже 5.2 и действующим модулем mod_rewrite. Необходимо распаковать скачанный архив в корневую директорию сайта. Через адресную строку браузера перейти в директорию системы http://{mysite}/textolite/. В дальнейшем директорию системы можно переименовать. Если все сделано правильно, система выведет форму авторизации. Пароль по умолчанию "admin". После первого входа пароль необходимо изменить в настройках системы.

Системе нужны права на запись для своего каталога и для всех файлов сайта. Чтобы не было проблем с отображением нелатинских символов, все редактируемые файлы должны быть в кодировке utf-8.

Для запуска системы под Nginx без Apache нужно настроить сервер по параметрам из файла .htaccess. Файл .htaccess в этом случае можно удалить. Для стандартной установки можно использовать следующий код: index textolite.php;
if (!-e $request_filename) {rewrite ^/textolite/(.*)$ /textolite/textolite.php?query=$1}
location ~ \.(ini|log)$ {deny all;}

Расширенная версия

Расширенная версия имеет несколько дополнительных возможностей. Можно удалять, клонировать и перемещать редактируемые элементы в визуальном редакторе. Можно заменять существующие картинки перетаскиванием файла в окно браузера. Попробовать эти функции можно в демоверсии на сайте.

Расширенная версия активируется в настройках стандартной версии специальным кодом. Получить код активации можно в автоматическом режиме поддержав проект на сумму $10. Тем, кто уже поддерживал проект ранее на любую сумму, достаточно связаться с нами по почте.

После первой активации код будет привязан к домену, на котором произошла активация. Количество доменов для одного кода зависит от суммы платежа. В пределах домена можно активировать неограниченное число копий Textolite. Доступ к расширенной версии остается навсегда, все последующие обновления будут доступны в автоматическом режиме.
